Odoo 14Journal Sequence

Bring backs the journal sequence concepts from Odoo 13. Now user can easily configure and change the invoice/bill sequence number from the journals. Download the Module and Install it.

Features

  • Entry and Refund Sequence For Journals.
  • Customize sequence as per need.
  • Works as in Odoo 13.
  • Overview
  • Screenshots

Overview

Journal sequence concept has been removed in Odoo 14. Thus now it is difficult for users to change the sequence of invoices and bills in Odoo 14. This app will bring the concept from Odoo 13, where user can set and change the sequence as it was in Odoo 13.
